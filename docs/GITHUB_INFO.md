## 📥 Como Clonar e Usar

### Opção 1: Git CLI (Terminal)

```bash
# Clonar repositório
git clone https://github.com/hansufsm/algoritmos-programacao-fisica.git

# Entrar no diretório
cd algoritmos-programacao-fisica

# Ver histórico
git log --oneline

# Ver remote
git remote -v
```

### Opção 2: GitHub Desktop
1. Acesse: https://github.com/hansufsm/algoritmos-programacao-fisica
2. Clique em "Code" → "Open with GitHub Desktop"
3. Escolha local para clonar

### Opção 3: Download ZIP

2. Clique em "Code" → "Download ZIP"

## 🎯 Como Fazer Contribuições

### Passo 1: Fork
```bash
# Vá para https://github.com/hansufsm/algoritmos-programacao-fisica
# Clique em "Fork" (canto superior direito)
```

### Passo 2: Clone seu Fork
```bash
git clone https://github.com/SEU-USUARIO/algoritmos-programacao-fisica.git
cd algoritmos-programacao-fisica
```

### Passo 3: Crie uma Branch
```bash
git checkout -b feature/sua-contribuicao
```

### Passo 4: Faça Mudanças
- Edite arquivos
- Teste localmente
- Commite com mensagem clara

### Passo 5: Push
```bash
git push origin feature/sua-contribuicao
```

### Passo 6: Pull Request
1. Vá para: https://github.com/hansufsm/algoritmos-programacao-fisica
2. Clique em "Compare & pull request"
3. Descreva suas mudanças
4. Envie!
