Perguntas Frequentes

O que é um algoritmo?

Um algoritmo é uma sequência finita, ordenada e bem definida de instruções para resolver um problema específico. É independente de linguagem de programação.

Quando usar PARA vs ENQUANTO?

Use PARA quando o número de repetições é conhecido (ex: preencher 10 linhas). Use ENQUANTO quando é desconhecido (ex: ler valores até que usuário diga "parar").

Qual é a fórmula do período de um pêndulo simples?

T = 2π√(L/g), onde L é o comprimento e g é a aceleração gravitacional. Não depende da massa ou amplitude (para pequenos ângulos)!
