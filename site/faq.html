Conceitos Gerais

O que é um algoritmo?

Um algoritmo é uma sequência finita, ordenada e bem definida de instruções para resolver um problema específico. É independente de linguagem de programação.

Qual a diferença entre Algoritmo, Pseudocódigo e Código?

Algoritmo Ideia em português natural Entendida por humanos
Pseudocódigo Portugol (VisualG) Formais, próximos de código
Código Python, Java, C++ Executável pelo computador

O que é complexidade Big-O?

Big-O mede como o tempo de execução cresce com o tamanho da entrada (n). Exemplos: O(1) constante, O(n) linear, O(log n) logarítmica, O(n²) quadrática.

Módulo 1: Fundamentos da Programação

O que é um fluxograma?

Um fluxograma é uma representação visual de um algoritmo usando símbolos padrão: óval (início/fim), retângulo (processo), losango (decisão), paralelogramo (entrada/saída).

Como fazer validação de dados?

Validação verifica se a entrada está dentro de limites aceitáveis usando estruturas condicionais:

se (idade >= 0) e (idade <= 120) entao
  escreva("Idade válida")
senao
  escreva("Idade inválida")
fimse

Quais são os tipos de dados em VisualG?

  • INTEIRO: números inteiros (-5, 0, 42)
  • REAL: números decimais (3.14, -0.5)
  • CADEIA: texto ("Olá", "FSC1189")
  • LOGICO: verdadeiro ou falso
Módulo 2: Estruturas de Iteração

Quando usar PARA vs ENQUANTO?

PARA: quando o número de repetições é CONHECIDO (ex: preencher 10 linhas)

ENQUANTO: quando é DESCONHECIDO (ex: ler valores até que usuário diga "parar")

O que é convergência de série?

Convergência significa que conforme adicionamos mais termos de uma série, o resultado aproxima-se cada vez mais de um valor limite. Exemplo: série de Leibniz π = 4(1 - 1/3 + 1/5 - ...)

Como somar elementos de um vetor?

soma <- 0
para i de 1 ate tamanho faca
  soma <- soma + v[i]
fimpara
escreva("Soma: ", soma)
Módulo 3: Funções e Métodos Numéricos

Qual a diferença entre função e procedimento?

Função: retorna um valor (usa retorne)

Procedimento: não retorna, apenas executa ações

O que é o método da bisseção?

Método para encontrar raízes de f(x)=0 dividindo repetidamente um intervalo [a,b] até convergir. Sempre funciona se f(a)·f(b) < 0.

Qual método de integração é mais preciso?

Método Erro Uso
Retângulo O(h) Mínimo
Trapézio O(h²) Comum, rápido
Simpson O(h⁴) Mais preciso
Módulo 4: Aplicações Avançadas

Qual é a fórmula do período de um pêndulo simples?

T = 2π√(L/g)

Onde L é comprimento e g é aceleração gravitacional. Não depende da massa ou amplitude!

O que é regressão linear?

Ajuste de uma reta y = ax + b aos dados para estimar relação linear. Usa método de mínimos quadrados. R² mede qualidade do ajuste (0 a 1).

Quais são os 3 regimes do oscilador amortecido?

  • Subamortecido (β < ω₀): oscila com amplitude decrescente
  • Criticamente amortecido (β = ω₀): retorna sem oscilar (menor tempo)
  • Superamortecido (β > ω₀): retorna lentamente sem oscilar
